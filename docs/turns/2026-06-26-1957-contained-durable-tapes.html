Contained Durable Tape Modules

Durable tape modules now own a bounded scroll viewport by default, so a page can mount many modules without the TanStack virtual spacer expanding the module, page, and DOM work.

Issue

islandflow-to8b

Surface

apps/web durable tapes, options, alerts, QA route

Production Recovery

Rebuilt and restarted only islandflow-web.service after a partial build state.

Findings

Root cause

DurableTape used @tanstack/react-virtual, but embedded tapes did not always have a bounded scroll container. The virtual body height then became page height, so the virtualizer rendered far too many rows.

Fix

Shared tape CSS now gives every tape a default contained height, while known full-height shells opt into their parent height. QA preview frames also have explicit heights.

Efficiency path

Options and alerts modules can consume server-composed durable row models and still honor local scope/filter behavior before rows reach the virtual list.
