Lightweight charts phase 05: hover readout and visual hardening

Added a compact crosshair readout, pure hover aggregation transforms, terminal-owned flow context wiring, rounded directional lower-pane bars, and responsive tooltip hardening. The chart module remains reusable; terminal state, smart-flow copy, and legacy smart-money compatibility stay in the terminal adapter. Reviewer repair tightened option-side direction, duplicate hover state, terminal fetch-state churn, and rounded-bar color rendering.

Changes Made

  • Added pure hover aggregation helpers for candle OHLC, volume, trade count, directional option notional, smart-flow evidence context, compatibility fallback, abstention, and why-not rows.
  • Added focused fixture coverage for bucket filtering, neutral and unknown notional, smart-flow preference, smart-money fallback, confidence, evidence quality, and row ordering.
  • Added MarketChartTooltip, a dense crosshair readout that renders stable core rows first and extension-owned rows under labelled groups.
  • Added a lightweight-charts custom series renderer for rounded directional lower-pane bars while keeping hover hit testing in the existing bucket model.
  • Wired terminal option prints, flow packets, smart-flow projections, and legacy smart-money context through adapter-owned hover providers without importing terminal code into features/market-chart.
  • Hardened tooltip placement and CSS for desktop dashboard, mobile, and compact embed sizes, with labels and text carrying directional meaning instead of relying on red or green alone.
  • Reviewer repair: made option notional direction call/put aware, deduped identical hover and live candle state frames, and rendered lower-pane custom bars with the effective lightweight-charts item color.

Implementation Notes

  • The reusable chart module exposes generic hover rows and context. The terminal adapter owns option-side mapping, smart-flow labels, and legacy compatibility language.
  • Core readout rows are stable: time with interval, OHLC, volume, and trades. Flow rows append under Flow context; lower-pane and overlay rows append under their own groups.
  • Directional option notional always renders bullish, bearish, and neutral or unknown rows so uncertain flow remains visible.
  • The rounded-bar renderer uses custom-series pane data and signed values. Positive values round the top corners, negative values round the bottom corners, and neutral values use a dim neutral treatment.
  • Tooltip placement flips around the crosshair on wider charts and uses a bounded full-width treatment on narrow charts, avoiding active-candle coverage when practical without layout shift.
  • Hover snapshots are signature-deduped before React state updates, and terminal chart fetch state ignores value-identical candle/status frames from live updates.
