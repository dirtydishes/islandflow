Summary

The new apps/web/features/durable-tape module owns the reusable feature resolver, responsive template selection, column helpers, row keys, hot/history composition, cursor selection, scroll-hold reducer, TanStack virtual list wrapper, and flat tape UI primitives.

Existing terminal helpers in terminal/tape.ts and terminal/scroll.ts now forward to the shared foundation, preserving the old exports and behavior while giving later domain phases a stable import target.

Scope Held

  • No rebuild of /options, /news, dashboard, alerts, equities, or domain callback contracts.
  • No API, storage, filter, or ClickHouse contract changes.
  • Equities dashboard behavior was preserved as summary/chart consumption, not converted into a visible scroll-held tape.
  • Web-only QA did not start the full local stack. If browser QA is needed later, use bun run dev:web, which is the exact script present in this repo.

+export const mergeNewestWithOverflow = (incoming, existing, limit, onTrim) => {
+  const combined = [...incoming, ...existing];
+  const seen = new Set();
+  const deduped = [];
+  for (const item of combined) {
+    const key = getDurableTapeItemKey(item);
+    if (seen.has(key)) continue;
+    seen.add(key);
+    deduped.push(item);
+  }
+  deduped.sort(compareDurableTapeNewestFirst);
+  const evicted = deduped.slice(limit);
+  if (evicted.length > 0) onTrim?.(evicted.length);
+  return { kept: deduped.slice(0, limit), evicted };
+};
+export const selectOlderHistoryCursor = (items, getCursor) => {
+  let oldest = null;
+  for (const item of items) {
+    const cursor = getCursor(item);
+    if (!oldest || compareDurableTapeOldestFirst(cursor, oldest) < 0) oldest = cursor;
+  }
+  return oldest;
+};
