Fix Synthetic 1m Candle Playback

Changed synthetic live playback so deterministic phase-03 fixtures inject on a slower demo cadence while SPY and focus symbols continue producing regular quote and print background data for normal 1m candle aggregation.

Branchlavender/fix-1m-candles
Beadsislandflow-aay8
ScopeSynthetic playback, live chart state guard

Summary

The chart renderer was left alone. The fix separates continuous synthetic market background traffic from deterministic demo fixture injection, so repeated full SPY fixture paths no longer get packed into every short live tick. Fixture events still keep their phase03-x:live:<serial>: trace prefixes and the shared projection helper strips label-only fields before events leave the adapter boundary.

Changes Made

  • Added SYNTHETIC_DEMO_RUN_INTERVAL_MS, scaleSyntheticDemoRunIntervalMs, and projectSyntheticDemoLiveEvent in packages/synthetic-market.
  • Restored continuous equity background quotes and prints on the regular emit cadence, always including SPY plus current focus symbols.
  • Kept the regular equity background loop lit-only; off-exchange synthetic prints now come from the explicit dark path so overlays and dark inference do not receive contradictory venue metadata.
  • Restored continuous dynamic option bursts on the regular emit cadence.
  • Moved deterministic phase-03 fixture injection to an immediate startup injection plus the slower demo-run cadence, with load profile volume controlling runs per injection.
  • Added a live-session reset guard for equity-candles and equity-overlay subscription-key changes.
