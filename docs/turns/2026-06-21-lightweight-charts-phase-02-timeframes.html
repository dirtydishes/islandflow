Lightweight Charts Phase 02 Timeframes

Implemented the reusable timeframe registry, favorite interval reducer and persistence, conservative terminal toolbar integration, 15m candle defaults, and manifest coverage for islandflow-mloi.2.

Scope

The work stayed inside Phase 02. The mounted change is limited to the existing dashboard chart pane controls; it does not replace the dashboard chart with the reusable module or move transport/fetching into features/market-chart.

Changes

  • Added transforms/timeframes.ts with the canonical 1m, 5m, 15m, 30m, and 1h registry.
  • Made 1m, 5m, and 15m default favorites, with a pure reducer for favorite, unfavorite, toggle, and reset actions.
  • Added versioned localStorage helpers that fall back to defaults for malformed persisted data.
  • Updated the existing chart toolbar with favorite buttons, a native dropdown, disabled unavailable intervals, and a selected-interval favorite toggle.
  • Normalized live candle subscriptions through the shared registry, so 15m is represented and unregistered intervals clamp to 1m.
  • Changed candle service defaults and local env examples to 60000,300000,900000.
