Typed Smart-Flow Alert Evidence Resolver

Added a typed server-side resolver for smart-flow alert evidence refs and moved the alert drawer from prefix inference to resolved evidence items.

Implementation

  • Added SmartFlowAlertEvidenceBundle, item variants, request schema, and the /lookup/smart-flow-alert-evidence path constant in @islandflow/types.
  • Implemented batched API resolution for flow packets, option prints, exact option NBBO, exact equity quotes, equity prints, synthetic labels, external context, and explicit unresolved refs.
  • Added bounded storage helpers for exact (option_contract_id, ts), exact (underlying_id, ts), and equity print trace lookups.
  • Wired the lookup route into API routing, lookup rate-limit classification, public route checks, and README route documentation.
  • Changed alert drawer hydration to use one typed bundle POST and render resolved NBBO, equity quote/print, synthetic label, external context, and unresolved rows explicitly.

Behavior Notes

The resolver preserves one output item per input ref in input order, while de-duping only the storage calls. option-nbbo:<contract>:<ts> and equity-quote:<underlying>:<ts> parse by the last colon so identifiers can contain colons.

Bare refs and print:* refs remain option-print trace ids for current provider traces. Semantic refs that cannot be parsed or found become typed unresolved rows with malformed_ref, not_found, or unsupported_ref.

+export const parseSmartFlowAlertEvidenceRef = (rawRef: string): ParsedEvidenceRef => {
+  const ref = rawRef.trim();
+  if (ref.startsWith("option-nbbo:")) {
+    const parsed = parseLastColonLookup(ref, "option-nbbo:");
+    return parsed
+      ? { kind: "option_nbbo", ref, lookup: { option_contract_id: parsed.left, ts: parsed.ts } }
+      : { kind: "unresolved", ref, inferredKind: "option_nbbo", reason: "malformed_ref" };
+  }
+  if (ref.startsWith("equity-quote:")) {
+    const parsed = parseLastColonLookup(ref, "equity-quote:");
+    return parsed
+      ? { kind: "equity_quote", ref, lookup: { underlying_id: parsed.left, ts: parsed.ts } }
+      : { kind: "unresolved", ref, inferredKind: "equity_quote", reason: "malformed_ref" };
+  }
+  if (ref.includes(":") && !ref.startsWith("print:")) {
+    return { kind: "unresolved", ref, inferredKind: "unknown", reason: "unsupported_ref" };
+  }
+  return { kind: "option_print", ref, traceId: ref };
+};
