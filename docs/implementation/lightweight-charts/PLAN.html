Visual Lane

Restrained

The chart should feel like an instrument panel inside Market Command: compact, flat, stable under live updates, and precise. The module should inherit terminal tones by default and avoid card framing.

bullish bearish neutral/info active control

Red and green are never the only direction signal. Readouts, legends, labels, or position must also carry meaning.

Extension Points

No forks
  • Price renderers: candles, Heikin Ashi, future bar or line modes.
  • Lower panes: smart direction, all flow, volume, future indicators.
  • Overlays and markers with adapter-owned payloads.
  • Toolbar slots for future actions without dashboard rewrites.
  • Settings sections and hover rows owned by data layers.
  • Layout presets for dashboard, full, compact, and replay embeds.

Subagent Use

Bounded lanes
  • Inventory current chart imports and CSS selectors.
  • Compare lightweight-charts v4 and v5 APIs used here.
  • Draft transform fixtures for Heikin Ashi and hover buckets.
  • Run browser visual QA across chart sizes.
  • Audit copy for overconfident smart-money language.

The main agent owns final API decisions, edits, Beads updates, commits, and pushes.

Non-Goals

Keep scope clean
  • No new scoring policy.
  • No historical calibration.
  • No new synthetic foundations.
  • No claim of hidden participant identity.
  • No dashboard-only chart API.
  • No card wrapper as the default module surface.
