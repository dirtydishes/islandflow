Phase 05 evidence record

Server-composed durable-tape row view models

Phase 05 introduces a typed row model for the busiest durable-tape lanes, composes option support and alert evidence summaries in the API process, and lets the web route consume composed rows when available while keeping the raw rendering fallback explicit.

Issue islandflow-ze79.6
Branch lavender/ze79-phase-05-server-composed-view-models
Probe Output baselines/phase-05-local-hosted-api.json
Delivery Path Websocket durable-rows snapshot and deltas

Scope

  • Added shared option and alert durable row view-model schemas in packages/types with typed cells, badges, support summaries, evidence summaries, and drilldown refs.
  • Added the live durable-rows subscription channel with lane, filter, underlying, contract, and snapshot limit support.
  • Composed option print support and alert evidence server-side from cached API live windows, including bounded snapshot/event composition.
  • Migrated the durable-tapes options and alerts panes to server rows when present, with dense raw panes left as the hosted-API fallback.

Contract Notes

The contract is lane-specific rather than a generic display blob. Option rows carry contract, execution, NBBO, signal, packet, classifier, smart-money, cells, badges, and drilldown refs. Alert rows carry alert identity, severity, direction, evidence counts, primary packet preview, option-print previews, unresolved refs, cells, badges, and drilldown refs.

The web route never needs packet, NBBO, classifier, or trace joins for normal server-row rendering. When the hosted API rejects durable-rows, the client sends the new subscription separately so older backends do not reject the raw durable subscriptions in the same batch.

Risks

  • The hosted API used for the required probe does not yet accept durable-rows. This PR verifies server-row composition through API tests and verifies browser fallback against hosted data. A deployed backend probe should confirm data-row-source="server" after rollout.
  • The server composition is bounded to the options and alerts lanes for Phase 05. Flow, equities, and news remain on their existing durable pane paths.
