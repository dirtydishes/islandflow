Durable-Tapes Performance Phase 00 Probe

Added the repeatable browser/CDP probe for /durable-tapes, recorded local hosted-API and deployed web baselines, and documented the first budget profile for islandflow-ze79.1. This phase intentionally changes no product runtime behavior.

Scope

Phase 00 is a feedback-loop PR. It creates a probe that later phases can run before and after behavior changes. The branch does not modify terminal state, hydration behavior, API handlers, storage queries, or route rendering.

Changes

  • Added scripts/probes/durable-tapes-perf.ts, a Bun script that launches Chrome/Chromium or connects to an existing CDP endpoint.
  • Collected network totals, support and by-trace request counts, aborted request counts, endpoint status distributions, websocket frames/bytes, CDP timing and heap metrics, DOM nodes, visible pane count, and per-pane row counts.
  • Defined the durable-tapes-3m-v1 budgets that fail on the current request storm while still requiring five visible durable panes and live row sanity.
  • Recorded baseline JSON artifacts for local web against the hosted API and the deployed web route.
  • Updated the phase spec and implementation index with the command, browser fallback, budget profile, and baseline summary.

Known Risks

  • The hard row budget is aggregate because individual live channels can be quiet. The report still records per-pane rows for reviewer judgment.
  • Budgets are tuned to catch the current failure mode, not to prove final low-power device readiness. Later phases should adjust them only with fresh evidence.
  • The probe depends on Chrome/Chromium or an externally supplied CDP endpoint; the phase doc records the fallback path.

+const attachCollectors = (client: CdpClient, getWindow: () => MetricWindow): void => {
+  client.on("Network.requestWillBeSent", (params) => {
+    const event = params as {
+      requestId: string;
+      request: { url: string; method: string };
+    };
+    const endpoint = classifyEndpoint(event.request.url);
+    const window = getWindow();
+    window.totalNetworkRequestCount += 1;
+    if (endpoint === "optionsSupport") {
+      window.optionsSupportRequestCount += 1;
+    }
+    if (endpoint === "optionPrintsByTrace") {
+      window.optionPrintsByTraceRequestCount += 1;
+    }
+    window.requestsById.set(event.requestId, {
+      url: event.request.url,
+      method: event.request.method,
+      endpoint
+    });
+  });
+  client.on("Network.loadingFailed", (params) => {
+    const event = params as { requestId: string; canceled?: boolean; errorText?: string };
+    const isAbort = event.canceled === true || event.errorText === "net::ERR_ABORTED";
+    if (!isAbort) return;
+    const window = getWindow();
+    window.abortedRequestCount += 1;
+    if (window.requestsById.get(event.requestId)?.endpoint) {
+      window.abortedEndpointRequestCount += 1;
+    }
+  });
+};

+const evaluateSanity = async (client: CdpClient): Promise<SanitySnapshot> => {
+  const expression = String.raw`

+    ["flow", ".durable-tapes-flow"],

+    ["news", ".durable-tapes-news"]

+  const paneRows = paneSpecs.map(([key, selector]) => {
+    const pane = document.querySelector(selector);
+    const rows = pane
+      ? Array.from(pane.querySelectorAll(".durable-tape-row")).filter(isVisible)

+  });
+  return {
+    routePresent: Boolean(document.querySelector('[data-route-variant="durable-tapes"]')),
+    visibleDurablePaneCount: paneRows.filter((pane) => pane.visible).length,
+    visibleRowCount: paneRows.reduce((sum, pane) => sum + pane.rowCount, 0),
+    paneRows
+  };
+})()
+`;

+    expression,
+    returnByValue: true,
+    awaitPromise: true
