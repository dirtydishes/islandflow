API Private Edge Phase 02 Same-Origin Production Transport

Phase 02 for islandflow-hnbk.3 switches production browser REST and websocket defaults to app-origin paths while keeping local development on the local API and preserving NEXT_PUBLIC_API_URL as an explicit override.

Outcome

Browser transport

Added a shared web transport helper and moved terminal, options, equities, flow packet, alerts, and news builders onto one same-origin/local/override contract.

Server-only API origin

Synthetic admin route proxy configuration now reads ISLANDFLOW_INTERNAL_API_URL instead of NEXT_PUBLIC_API_URL.

Edge matcher

NPM same-origin matcher and public route checks now cover ws, replay, prints, joins, nbbo, quotes, dark, flow, candles, history, news, lookup, option-prints, and equity-joins.

Probe posture

Durable-tapes production smoke now fails if the scanned browser bundle contains the supplied raw or internal API origin.
