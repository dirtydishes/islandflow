Implementation Notes

  • Added FlowPacketsTape as the domain wrapper over the shared DurableTape foundation.
  • Moved packet summary logic into explicit helpers for contract, member count, premium/notional, window, side, quote state, and quality labels.
  • Added the shared packet-focus contract in apps/web/features/flow-packets/types.ts: FlowPacketFocusRequest and OnPacketFocus.
  • Kept packet focus as a typed callback. The module does not import terminal drawer state or mutate route-specific terminal focus state.
  • Changed FlowPane into a compatibility adapter that passes terminal filters/scope into FlowPacketsTape.
  • Added explicit stale/missing quote text badges so warning state is not color-only.

Scope Held

  • No options packet-print table behavior was added.
  • No alert evidence hydration was changed.
  • No new packet scoring or classifier policy was introduced.
  • No durable-tape public interface churn was needed.
