Summary

The new apps/web/features/equities-tape package exports EquitiesTape, domain columns, templates, formatting, filter helpers, and a default source adapter backed by /ws/live equity subscriptions and /history/equities paging.

The module keeps joins, dark context, chart overlays, and route composition outside the package. Linked join or dark evidence is an optional render callback, while ticker focus and print inspect are explicit callbacks owned by the embedding route.

Scope Held

  • No market-chart, candle, overlay, or terminal-state imports were added.
  • No shared durable-tape public interface changes were needed.
  • No route or dashboard integration was committed. Browser QA used a temporary local harness that was removed before closeout.
  • Venue and off-exchange filters are applied in the module today because current equity history APIs only server-filter ticker scope.

Reviewer Repair

Review found one sparse-history pagination bug in the client-side venue and off-exchange filter path. When a bounded scan crossed only non-matching server pages, the source could return an empty item page while still holding a server cursor. The shared tape then had no row-count change to trigger another history request, or could mark the filtered stream exhausted too early.

The repair keeps the durable-tape public contract unchanged. DurableTape now remembers the returned nextCursor for the current query and includes cursor advancement in the history-gate signal. The equities source now reports exhaustion only when there is no next cursor. A regression test covers the sparse filtered cap case.

+export const loadEquitiesTapeHistoryPage = async ({ cursor, scope, filters, options }) => {
+  const fetcher = options?.fetcher ?? fetch;
+  let nextCursor = cursor;
+  for (let page = 0; page < maxPages && nextCursor; page += 1) {
+    const url = new URL(buildEquitiesTapeApiUrl("/history/equities", options?.apiBaseUrl));
+    url.search = getEquitiesTapeHistoryParams({ cursor: nextCursor, scope, filters, limit }).toString();
+    const payload = await parseHistoryResponse(await fetcher(url.toString()));
+    const filtered = filterEquityPrints(payload.data ?? [], filters);
+    if (filtered.length > 0 || !payload.next_before) {
+      return { items: filtered, nextCursor: payload.next_before, exhausted: !payload.next_before };
+    }
+    nextCursor = payload.next_before;
+  }
+  return { items: [], nextCursor, exhausted: !nextCursor };
+};
+
+export const createEquitiesTapeSource = (options = {}) => ({
+  subscribe: ({ scope, filters }) => {
+    const subscription = getEquitiesTapeSubscription(scope ?? {}, snapshotLimit);
+    const subscriptionKey = getLiveSubscriptionKey(subscription);
+    const socket = new WebSocket(buildEquitiesTapeWsUrl("/ws/live", options.wsBaseUrl));
+    socket.onopen = () =>
+      socket.send(JSON.stringify({ op: "subscribe", subscriptions: [subscription] }));
+    socket.onmessage = (event) => {
+      const message = JSON.parse(event.data);
+      if (message.op === "snapshot" && getLiveSubscriptionKey(message.snapshot.subscription) === subscriptionKey) {
+        replaceItems(parseEquityPrints(message.snapshot.items));
+      }
+      if (message.op === "event" && getLiveSubscriptionKey(message.subscription) === subscriptionKey) {
+        mergeItems([parseLivePayload("equities", message.item)]);
+      }
+    };
+  },
+  loadOlder: (cursor, query) => loadEquitiesTapeHistoryPage({ cursor, ...query, options })
+});
