Summary

The new apps/web/features/options-tape module owns options contract display, filter presets, responsive templates, row rendering, hover detail, history loading, and the packet-focus callback type. The route now composes OptionsTape as the single tape-first body of /options, while the terminal state remains the live transport and support-context source.

Packet focus is represented as an exported domain callback: FlowPacketFocusRequest and OnPacketFocus in apps/web/features/options-tape/types.ts. It uses FlowPacket.id, FlowPacket.members trace IDs, and features.option_contract_id when available.

Scope Held

  • No flow-packets module rebuild, alert rebuild, classifier semantic change, ingestion change, or ClickHouse schema change.
  • Legacy terminal drawers are not opened by options row activation. Packet and contract scope stay inside the options tape.
  • The existing /tape compatibility redirect to /options is preserved.
  • Phase 03 still owns packet summary, packet detail, packet row rendering, and packet inspect surfaces.

Implementation Notes

  • Default filters are view: "signal", sides AA/A/MID, calls plus puts, and stock security type.
  • Global signal columns are TIME | CONTRACT | PX | SIZE | PREMIUM | SIDE | IV.
  • Packet focus columns are DT | TIME | PX | SIZE | PREMIUM | SIDE | SPOT.
  • Contract focus columns are TIME | PX | SIZE | PREMIUM | NBBO | SIDE | EXCH | IV.
  • Settings include view, side presets, type presets, security presets, premium presets, custom premium, and reset.
  • Hover detail exposes raw contract ID, trace ID, exchange, conditions, NBBO, quote age, IV source, signal profile, reasons, and packet ID.

+const activatePrint = (event) => {
+  const { print, packet } = contextForPrint(event.item);
+  if (packet?.packet) {
+    setScope(getPacketFocusScope(print, packet.packet));
+    onContractFocus?.(print);
+    onPacketFocus?.(packetRequestFromPrint({ print, packet: packet.packet, source: "options-tape" }));
+    return;
+  }
+  setScope(getContractFocusScope(print));
+  onContractFocus?.(print);
+};
