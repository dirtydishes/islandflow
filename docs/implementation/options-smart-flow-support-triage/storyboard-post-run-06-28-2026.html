Every phase used a visible worker thread, a separate thermo-nuclear review thread, Forgejo PRs, and orchestrator-owned Beads closeout.

Implementation arc

From resolver to operator workflow

The loop kept support computation on the server, preserved compact browser contracts, then layered user intent: packet scope, exact contract scope, diagnostics, settings, and detail. The final follow-up closed the only discovered child issue by making unavailable history paths degrade locally instead of breaking route render.
