Parallelization Rules

Parallel lanes are explicit, not inferred.

Safe To Split

  • Phase 01: route/config, formatting, tape reducers, evidence helpers.
  • Phase 02: scroll/virtualization and live/session hooks.
  • Phase 04: OPRA, News, Dashboard/Chart, Drawers after phase 03.
  • Phase 05: cleanup by candidate after reachability is clear.

Keep Serial

  • Phase 03 owns the shared state and shell seam.
  • Phase 06 owns final verification and publishing.
  • No phase starts until bd ready shows it unblocked.

Scope Discipline

These constraints prevent accidental behavior changes.

Preserve

  • apps/web/app/terminal.tsx remains a compatibility facade until cleanup.
  • Supported routes: /, /options, /news, and /tape -> /options.
  • Replay data mode remains available.
  • Smart-money fallback remains unless a separate decision removes it.
