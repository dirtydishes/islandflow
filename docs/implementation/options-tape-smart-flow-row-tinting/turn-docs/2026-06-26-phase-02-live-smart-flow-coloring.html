Options Tape Smart-Flow Tinting Phase 02

Wired live smart-flow projections into OptionsTape for current live rows, including direct option-print evidence refs, known FlowPacket.members expansion, hover context, and focused packet or contract scope labels. Historical loaded-row completeness remains Phase 03.

Outcome

/options now subscribes to smart-flow through an explicit route feature without enabling the legacy smart-money subscription. Dashboard Recent Contracts, the OPRA route, and the durable-tapes OptionsTape fallback receive filtered live smart-flow projections.

OptionsTape builds a deterministic projection-to-print map locally in the options-tape domain. Direct option-print refs decorate those prints, packet refs expand through known FlowPacket.members, newer projections win on collisions, and direct print evidence wins over packet expansion for the same row. Smart-flow row tinting takes precedence over legacy classifier or smart-money decor.

Smart-flow hue is keyed by hypothesis type, direction remains a row modifier and context label, policy confidence controls intensity, and abstention lowers intensity with explicit row context. Hover details and the packet or contract focus band show hypothesis, direction, confidence, and abstention state when smart-flow context is present.
